# 英语课堂讲解 参考结构

## 目标
- 面向小学英语课堂，针对具体 Lesson 生成10分钟以内的教学课件
- 内容来源于教材，结构符合教学逻辑

## 推荐教学环节

### 1. 封面（1页）
- 课题名称（中英双语）
- 年级和课时信息
- 简洁装饰

### 2. Warm-up 暖场导入（1页）
- 与本课主题相关的趣味问题
- 用图片或情境激发学生兴趣
- 时间控制：约1分钟

### 3. 课文学习 Text Learning（1-2页）
- 完整课文展示
- 关键句子标注
- 中文翻译辅助
- 时间控制：约3分钟

### 4. 重点词汇 Key Vocabulary（1页）
- 本课新单词列表
- 音标/发音提示
- 中文释义
- 例句
- 时间控制：约2分钟

### 5. 重点句型 Key Sentences（1页）
- 本课核心句型结构
- 例句展示
- 替换练习引导
- 时间控制：约2分钟

### 6. 课堂练习 Practice（1页）
- 选择题 / 填空题 / 连线题
- 口语互动练习
- 时间控制：约1.5分钟

### 7. 课堂总结 Summary（1页）
- 本课知识点回顾
- 课后作业布置
- 时间控制：约0.5分钟

## 提示
- 优先保证课堂节奏感，每页停留时间适中
- 英文内容字号要大（22-28pt），确保后排学生可见
- 中文辅助信息字号适中（14-18pt）
- 颜色要鲜明活泼，适合小学生注意力
- 重点单词建议用醒目颜色（如蓝色或橙色）标注
